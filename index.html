  • MACHINE LEARNING

1) Hand Written Digit Classification:- MNIST dataset of handwritten digits is classified using Multilayer Perceptron Neural Network. A single hidden layer is used to classify the digits in the data set. Each pixel of an image is considered as the feature for the model. Weights for the hidden layer are learnt using back-propagation technique and regularization is implemented to avoid over fitting of the model. Accuracy of around 95% is achieved.

  • DISTRIBUTED SYSTEMS

1) Group Messenger :- An "Android" application which provides Decentralized Ordering with Failure Handling. Total and FIFO ordering constraints are maintained throughout all AVDs present in the system which is the core of the application, provided without using any master slave technique. Along with it failure handling of the various nodes in the system is taken care by using a 3-way request response mechanism.

2) Simple Distributed Hash Table:- Distributed Hash Table is based on CHORD which provide the 3 main functionalities of ID Space Partitioning, Ring Based Routing and Node Join. SHA-1, cryptographic hash function is at the core of this project which provides the basis for the routing of key value pairs in the system. Each node in the system knows about its predecessor as well as it successor to form a ring.

3) Replicated Key Value Storage :- A simplified version of Amazon Dynamo is implemented to understand how "Dynamo Style" key value storage works. The main goal of this project is to provide both "Availability" and "Linearizability" at the same time. In other words, the implementation always perform read and write operations successfully even under failures. At the same time, a read operation always return the most recent value. Replicated storage is used to provide availability and Chain Replication technique is used to provide linearizability.
